# CountDown

[CodesandBox](https://codesandbox.io/s/countdown-jwjfw)   

倒數計時器

* 範圍年月日時分秒

* 功能
  * 減速/加速 - 加速或減速倒數速度最高4倍最低0.25倍
  * -5秒/+5秒 - 加或減5秒數
  * 繼續/暫停 - 開始或暫停倒數
  * 重置 - 還原設定秒數
  * 清除 - 秒數規0
  * 重設 - 重新設定秒數
